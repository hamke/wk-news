1. 워드프레스의 검색 기능

워드프레스(WordPress)에는 검색 기능이 기본적으로 탑재되어 있습니다. 이러한 검색 기능을 이용하면, 워드프레스로 구축한 웹사이트의 포스트와 페이지를 손쉽게 검색할 수 있습니다.

하지만 때로 이러한 검색 결과에 원하지 않는 포스트 또는 페이지가 출력되는 경우가 있습니다.

이럴 때 운영 중인 워드프레스 웹사이트 내의 특정 글(포스트/페이지 등)을 검색 결과에서 간편하게 제외할 수 있는 방법이 있습니다.

2. 워드프레스 포스트 ID 파악하는 방법

먼저 검색 결과에서 제외하고 싶은 글의 ID를 확인합니다.

고유 주소(Permalinks) 메뉴에서 기본(Plain) 형태로 설정되어 있다면, 해당 페이지 주소(예:~/?p=123)에서 확인할 수 있습니다.

만약 기본(Plain) 형태가 아니라면, 어드민(관리자) 페이지에서 해당 글 링크에 마우스를 올려(Hover) 놓으면, 위와 같이 화면 아래 부분에서 포스트 ID를 확인할 수 있습니다.
