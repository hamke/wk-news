◼︎ 워드프레스 우커머스로 제작한 쇼핑몰 구축 사례 모음

워드프레스 우커머스(WordPress WooCommerce)로 자체 쇼핑몰/스토어 사이트를 오픈하거나, 유무형의 상품 판매 기능이 탑재된 웹사이트/홈페이지/블로그 등을 제작/구축하고자 할 때, 가장 먼저 하는 작업이 벤치마킹할 수 있는 기존 워드프레스 우커머스 쇼핑몰 사이트 정보를 찾는 일입니다.

하지만 막상 워드프레스 우커머스로 만들어진 쇼핑몰 사이트를 찾으려고 하면, 어디서 찾아야 할지 막막할 때가 많습니다.

워드프레스웹코리아(WP-KR.COM)가 워드프레스 우커머스(WooCommerce)로 제작된 국내/해외(한국/일본/중국/미국/유럽 등) 각 지역의 쇼핑몰 사이트 주요 구축 사례 및 예시 모음을 정리해서 소개해 드립니다. (계속 업데이트)

[참고] 워드프레스(WordPress)로 제작된 국내/해외 주요 구축 사례/예시 사이트 모음 (한국/일본/중국/미국/유럽 등)

[참고] 워드프레스 사이트 랜덤 뷰어 (WordPress Sites Random Viewer) : 워드프레스로 제작된 전세계 사이트 구축 사례 모음 간편 확인 서비스

[참고] 워드프레스 애널리틱스 (WordPress Analytics) : 사이트 정보(테마/호스팅/플러그인/도메인 등) 간편 확인/분석 서비스
