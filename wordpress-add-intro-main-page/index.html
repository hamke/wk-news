◼︎ 1. 워드프레스 메인(인트로) 페이지 추가 및 수정하는 방법

워드프레스(WordPress)를 기반으로 웹사이트를 운영하고자 한다면, 일반적으로 먼저 원하는 워드프레스 테마(Theme)를 선택하고 설치하고는 합니다.

설치한 워드프레스 테마를 기반으로 완성된 웹사이트의 디자인은 전반적으로 마음에 들 수 있지만, 때로는 테마가 제공하는 기본 페이지 템플릿과는 다른, 별도의 페이지를 추가하거나 수정하고 싶은 때가 있습니다.

특히 웹사이트의 얼굴 격인 메인 페이지(인트로, Intro)를 기본 테마와는 다른 별도의 템플릿 디자인으로 꾸미고 싶은 경우가 많습니다.

◼︎ 2. 워드프레스 초보자들이 흔히 하는 착각

워드프레스(WordPress) 구동 원리와 웹프로그래밍에 익숙하지 않은 초보자들은 나름 열심히 인터넷 검색을 해서, 일부 블로그나 유튜브 채널, 커뮤니티에서 알려주는 것과 같이 워드프레스 관리자(어드민) 대시보드의 “설정” 메뉴의 정적 페이지 부분을 변경을 해 보기도 하지만, 원하는 디자인으로 표시되지 않는 경우가 많습니다.

또한 일반적인 웹서버 상의 웹프로그래밍 문법에 따라, 워드프레스 설치 디렉토리 내의 index.php 파일을 수정하거나, index.html 파일을 그대로 추가해 보기도 하지만, 그러한 방식으로는 결코 원하는 결과를 얻을 수 없습니다.

워드프레스(WordPress)는 일종의 웹앱 프레임워크로서, 자체적으로 정의하고 있는 라우팅(Routing, 경로 설정) 규칙이 있으며, 얼핏 보면 간단해 보이지만, 워드프레스 프레임워크의 원리를 제대로 학습하지 않으면, 사이트 운영 중 발생하는 여러 가지 문제를 결코 간단하게 해결할 수 없는 것입니다.

(일부 워드프레스 관련 컨텐츠를 다루는 블로그/유튜브 채널 등에서는, 비전문가(주로 온라인 마케터, 비개발자 등)들이 워드프레스의 구동 원리를 제대로 이해하지 않은 채, 자신들의 경험담(뇌피셜) 위주의 엉뚱한 정보들 게시하여 일반 사용자들로 하여금 혼란을 가중시키기 때문에, 참고하는 워드프레스(WordPress) 관련 정보 출처에 대해서도 주의를 기울일 필요가 있습니다.)

◼︎ 3. 워드프레스 메인 페이지 자동 생성 플러그인 활용 방법

워드프레스 메인 페이지 자동 생성 플러그인 (WordPress Auto Main Page Generator)은 설치된 테마가 제공하는 기본 메인(인트로) 페이지와 별개로, 새로운 메인 페이지를 간편하게 추가/설정할 수 있는 워드프레스 플러그인입니다.

워드프레스 메인 페이지 자동 생성 플러그인 (WordPress Auto Main Page Generator)을 워드프레스 사이트에 설치한 뒤, 사이트의 메인(인트로) 페이지에 접속해 보면, 기존의 메인 페이지 대신 빈 페이지가 나타납니다.

워드프레스 어드민(관리자) 페이지의 “WP Auto Main Page” 메뉴를 클릭하면, HTML 코드를 입력/저장할 수 있는 화면이 나타납니다.

메인(인트로) 페이지에 설정하고자 하는 HTML 소스를 붙여놓기한 뒤 저장(Save) 버튼을 클릭합니다.

곧바로 사이트의 메인(인트로) 페이지에 접속해 보면, 위와 같이 원하는 디자인의 웹페이지가 보이는 것을 확인할 수 있습니다.

입력/저장해 놓은 HTML 소스 코드 내에서 원하는 텍스트 문구와 이미지, CSS 등을 수정한 뒤 다시 저장하면, 메인(인트로) 페이지의 내용이 간단하게 변경(업데이트)되는 것을 확인할 수 있습니다.

플러그인을 통해 기존 테마가 제공하는 인트로(메인) 페이지가 수정되거나 삭제되는 것이 아니기 때문에, 기존 메인 페이지 디자인으로 돌아 가고 싶을 경우, 플러그인을 비활성화하여 간편하게 다시 이전 메인 페이지로 돌아갈 수 있습니다.

워드프레스 구동 원리에 대해 익숙하지 않은 초보/입문자라면, 새로운 메인 페이지 추가 생성/설정이 필요할 때, 워드프레스 메인 페이지 자동 생성 플러그인 (WordPress Auto Main Page Generator)을 활용해서 간편하게 해결해 보세요.
