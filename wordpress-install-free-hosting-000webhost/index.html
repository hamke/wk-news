◼︎ 워드프레스 설치를 위해 반드시 필요한 호스팅 서비스

워드프레스(WordPress)를 설치하기 위해서는, 워드프레스 구동에 최적화된 웹서버 및 데이터베이스(Database) 환경이 필수적입니다.

이러한 워드프레스를 설치하고 운영할 수 있는 웹호스팅 서비스는 시중에 많이 있지만, 워드프레스웹코리아(WP-KR.COM)가 그 중에서도 비용에 대한 부담없이 워드프레스를 간편하게 설치할 수 있는 호스팅 서비스를 소개해 드립니다.

◼︎ 1. 무료 호스팅 서비스 회원 가입 방법

000웹호스트(000WEBHOST.COM)는 보급형 웹호스팅 서비스로 유명한 호스팅어(Hostinger)가 운영하는 무료 호스팅 서비스입니다.

무료 호스팅 서비스를 이용하기 위해서 먼저 000웹호스트(000WEBHOST.COM) 홈페이지에 접속합니다.

  • 000웹호스트 (000WEBHOST.COM) : 바로 가기

홈페이지 첫 페이지에서 우측 상단의 “무료 회원 가입(Free Sign Up)” 버튼을 클릭하고, 회원 가입 페이지로 이동합니다.

회원 정보 입력란에 사용자의 이메일과 비밀번호, 원하는 웹사이트명을 입력한 뒤, 확인(“Get Free Hosting”) 버튼을 클릭합니다.

회원 가입을 완료한 뒤, 이메일과 비밀번호 정보를 입력해서 로그인합니다.

정상적으로 로그인하고 나면, 어드민 대시보드 페이지에 현재 운영 중인 웹사이트 목록이 보입니다.

워드프레스 기반의 신규 웹사이트를 설치하기 위해, ‘+’ 버튼을 클릭합니다.

새로 설치하고자 하는 웹사이트의 이름(Website Name)과 비밀번호(Password)를 입력합니다.

이 때 입력하는 비밀번호는 원격 파일 전송을 위한 FTP(File Transfer Protocol)과 동일하게 적용됩니다.

◼︎ 2. 워드프레스 간편 설치 방법

신규 웹사이트를 위한 기본 정보를 입력하면, 웹사이트 설치 유형을 선택할 수 있는 창이 나옵니다.

설치 유형 중 “워드프레스 웹사이트 설치하기(Build WordPress Website)” 옵션을 클릭합니다.

워드프레스 관리자(어드민) 페이지 접속을 위한, 사용자명(Username)과 비밀번호(Password), URL(서브도메인 주소용), 표시 언어(Language)를 입력한 뒤, ‘설치(Install)’ 버튼을 클릭합니다.

필요한 정보가 정상적으로 입력되면, 워드프레스 설치가 시작됩니다.

워드프레스 설치가 완료되면, 위와 같은 메시지가 나옵니다. “설정 페이지로 가기(Go to Configuration Page)” 버튼을 클릭합니다.

설정 페이지(Configuration Page)로 이동하면, 신규로 생성된 워드프레스 웹사이트에 대한 기본 정보와 관련 데이터를 확인할 수 있습니다.

생성된 웹사이트의 URL 주소를 확인한 뒤, 어드민 대시보드 페이지(Admin Dashboard)에 접속하여, 워드프레스 웹사이트 운영을 위한 기본 세팅을 완료합니다.

◼︎ 워드프레스 추천 호스팅

000웹호스트(000WEBHOST)는 실제 서비스로 사용하기에는 속도 등에서 다소 부족한 스펙일 수 있지만, 무료라는 가격을 고려한다면 꽤 괜찮은 스토리지 용량과 대역폭(Bandwidth)을 제공하고 있습니다.

처음 워드프레스(WordPress)를 접하는 분이나, 온라인에서 테스트용으로 부담없이 사용하고자 하는 분들이 있다면, 한번 사용해 보시기를 추천 드립니다.
