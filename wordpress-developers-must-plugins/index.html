◼︎ 워드프레스 개발자가 알아두면 좋은 플러그인

워드프레스(WordPress)에는 프로그래밍에 대해 익숙하지 않은 일반 사용자들 뿐만 아니라, 워드프레스를 전문적으로 다루는 프로그래머를 위한 다양한 플러그인(Plugins)들도 개발되어 있습니다.

워드프레스톡(WP-TALK.COM)이 워드프레스 테마 및 플러그인을 커스터마이징하거나 사용 중 발생하는 여러 가지 에러를 디버깅할 때, 유용하게 활용할 수 있는 주요 플러그인을 소개해 드립니다.

◼︎ 1. Query Monitor

“Query Monitor“은 워드프레스가 작동할 때 발생하는 모든 데이터베이스 쿼리(Database Queries)에 관한 내용을 표시해 주는 플러그인입니다.

데이터베이스 쿼리 정보 뿐만 아니라, 훅(Hooks)과 PHP 에러, AJAX 및 REST API 상태 정보, 워드프레스 구동 환경 등에 대한 내용도 확인할 수 있습니다.

  • Query Monitor : 자세히 보기 및 다운로드

◼︎ 2. Theme Check

“Theme Check“은 워드프레스 테마에 대한 코딩 규칙과 가이드라인에 대해 테스트하고 사전 평가해 줍니다. 이러한 방식을 통해, 워드프레스의 공식 레포지터리(Repository)에게 거절(Reject)되는 것을 방지할 수 있습니다.

워드프레스 테마를 개발하거나 커스터마이징할 때 반드시 필요한 플러그인입니다.

  • Theme Check : 자세히 보기 및 다운로드

◼︎ 3. Debug Bar

“Debug Bar“은 데이터베이스 쿼리(Database Queries)와 캐시(Cache) 등 디버깅 정보를 보여줍니다. PHP 에러를 쉽게 추적할 수 있습니다.

  • Debug Bar : 자세히 보기 및 다운로드

◼︎ 4. Log Deprecated Notices

“Log Deprecated Notices“은 무효화(deprecated)된 함수 및 기능을 알려 줍니다. 무효화된 파일과 함수 등 올바르지 못한 사용 부분과 그에 대한 대안 방식 정보을 제공합니다.

만일 오래된 테마나 플러그인을 업데이트하고 싶거나, 최신 코딩 가이드라인을 적용하고 싶을 때 유용한 플러그인입니다.

  • Log Deprecated Notices : 자세히 보기 및 다운로드

◼︎ 5. What The File

“What The File“은 보고 있는 워드프레스 페이지가 어떤 파일을 통해 구성되어 있는지 보여주는 플러그인입니다. 테마와 템플렛 개발을 할 때 유용한 플러그인입니다.

  • What The File : 자세히 보기 및 다운로드

◼︎ 6. User Switching

“User Switching“은 사용자 계정을 편리하게 변경할 수 있는 기능을 제공하는 플러그인입니다. 다양한 계정의 권한에 대한 기능을 개발하고 확인하고자 할 때 매우 유용합니다.

어드민 권한을 가진 상태에서 사용자 메뉴(Users menu)에서 “Switch To” 버튼을 클릭하여 간편하게 해당 사용자 계정으로 변경할 수 있습니다.

  • User Switching : 자세히 보기 및 다운로드

◼︎ 7. Advanced CSS Editor

“Advanced CSS Editor“은 스크린 사이즈에 따라 편리하게 CSS 코드를 입력할 수 있는 기능을 제공하는 플러그인입니다.

커스터마이징 메뉴에서 손쉽게 CSS 코드를 입력하여 디자인을 변경할 수 있습니다.

  • Advanced CSS Editor : 자세히 보기 및 다운로드

◼︎ 맺음말

워드프레스톡(WP-TALK.COM)이 소개해 드리는 방법들을 통해, 워드프레스를 보다 편리하게 활용하는데 도움이 되기를 바랍니다.

워드프레스(WordPress) 사이트 개선 및 각종 문제 해결, 워드프레스 핵심 원리 교육, 코딩을 통한 온라인 업무 자동화(Task Automation) 등에 대한 문의 사항이 있으시면, 카카오톡으로 연락 주세요.

◼︎ 도움이 될 만한 연관 추천 글

  • “워드프레스 애널리틱스(WordPress Analytics)” : 외부 웹사이트/쇼핑몰 테마 및 플러그인 간편 자동 분석 방법

  • “워드프레스 세일 나우(WordPress Sale Now)” : 테마/플러그인/호스팅 할인 프로모션 정보 간편 확인 방법

  • 워드프레스 사이트 제작할 때 반드시 알아둬야 할 필수 온라인 툴 서비스 모음

  • 워드프레스로 제작된 사이트 최신 사례/예시 모음 (한국/일본/중국/미국/유럽 등)

  • 워드프레스 핵심 원리 강의 레슨 및 교육 강좌 문의 (1:1개인/그룹/기업, 년 월) - 워드프레스톡

  • 워드프레스 관련 각종 문제 및 오류(에러)를 스스로 해결할 수 없을 때 추천 방법

  • 워드프레스(WP) 초보자가 알아두면 좋은 추천 글 모음

  • 워드프레스(WP) 설치에 최적화된 추천 호스팅 서비스 모음

  • 워드프레스 차일드(자식) 테마를 무료 온라인 빌더로 간편하게 자동 생성하는 방법

  • 워드프레스 실시간 외부 컨텐츠를 내 웹사이트로 가져 오는 방법 (REST API)

  • 워드프레스(WP) 책으로 보다 체계적으로 학습하기 - 쿠팡(웹/모바일앱)

  • 최신 노트북/컴퓨터 추천 상품 목록 보러 가기 - 쿠팡(웹/모바일앱)

워드프레스톡(WP-Talk)은 워드프레스(WP) 웹사이트에 대한 문제가 있을 시, 문제를 신속하게 진단하고 해결해 드립니다. 서비스에 대한 문의 사항이 있으시면, 카카오톡으로 연락 주세요.
